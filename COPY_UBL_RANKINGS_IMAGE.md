# Instructions: Copy UBL Rankings.png to Public Folder

To make the background image work on the Rankings page, you need to copy the file:

**FROM:** `tantalus-boxing-club/src/UBL Rankings.png`  
**TO:** `tantalus-boxing-club/public/UBL Rankings.png`

After copying, the Rankings page will automatically display the background image.

If the file doesn't exist in `src/`, please:
1. Locate the `UBL Rankings.png` file
2. Copy it to `tantalus-boxing-club/public/UBL Rankings.png`
3. The code is already configured to use `/UBL Rankings.png` from the public folder
