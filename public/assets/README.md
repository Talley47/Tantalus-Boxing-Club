# Assets Directory

## Logo Files

Place your logo files in this directory:

- `Undisputed Logo.png` - Main logo for the Tantalus Boxing Club application

## Usage

The logo is referenced in the TBCLogo component at `/assets/Undisputed Logo.png`

Make sure to:
1. Add your `Undisputed Logo.png` file to this directory
2. The file should be optimized for web use (recommended size: 80x60px or similar)
3. Use PNG format for best quality with transparency support
